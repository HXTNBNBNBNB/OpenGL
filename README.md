    3).介绍着色器  着色器是运行在gpu上的程序 类似c++程序运行在cpu上
        等于我们告诉gpu需要做什么  因为和图形有关的事 gpu运行起来更快
        着色器 主要流行两种类型   顶点着色器 和 片段（像素）着色器
        有多少个顶点就会调用多少次顶点着色器
        片段着色器的工作是为每个像素上色 有多少像素点就调用多少次
